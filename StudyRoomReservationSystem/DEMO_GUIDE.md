# StudyRoomReservationSystem Demo Guide

## 课堂演示目标

这份演示说明用于课堂汇报时快速展示系统主流程，覆盖：

- 学生登录
- 查看自习室座位状态
- 预约座位
- 阻止重复预约
- 查看我的预约
- 取消预约
- 管理员查看全部预约
- 管理员查看统计信息

## 演示前准备

在项目根目录编译并运行程序。

### MSVC

在 `Developer Command Prompt for VS` 中运行：

```bat
cl /EHsc /std:c++17 /utf-8 main.cpp User.cpp Student.cpp Admin.cpp Seat.cpp StudyRoom.cpp Reservation.cpp SystemManager.cpp /Fe:app.exe
app.exe
```

### g++（可选）

```bash
g++ -std=c++17 main.cpp User.cpp Student.cpp Admin.cpp Seat.cpp StudyRoom.cpp Reservation.cpp SystemManager.cpp -o app
./app
```

## 推荐演示流程

### 第一部分：学生端

1. 在主菜单输入 `1`
   - 进入学生登录
2. 输入账号 `stu001`
3. 输入密码 `123456`
   - 预期：显示学生菜单

4. 输入 `2`
   - 查看指定时间段座位状态
5. 输入房间号 `A101`
6. 输入时间段编号 `5`
   - 预期：`A101` 的 `3` 号座位显示为已预约

7. 输入 `3`
   - 进入预约
8. 输入房间号 `A101`
9. 输入座位号 `3`
10. 输入时间段编号 `5`
   - 预期：提示预约失败，因为该座位同一时间段已被占用

11. 再次输入 `3`
12. 输入房间号 `A101`
13. 输入座位号 `4`
14. 输入时间段编号 `5`
   - 预期：预约成功

15. 输入 `4`
   - 查看我的预约
   - 预期：能看到 `A101` 的 `3` 号和 `4` 号座位预约记录

16. 输入 `5`
   - 取消预约
17. 选择刚刚新增的那条记录编号
   - 推荐取消第 `2` 条
   - 预期：取消成功，只删除当前登录学生自己的预约

18. 输入 `0`
   - 退出学生端

### 第二部分：管理员端

1. 回到主菜单后输入 `2`
   - 进入管理员登录
2. 输入账号 `admin`
3. 输入密码 `123456`

4. 输入 `1`
   - 查看全部预约
   - 预期：能看到系统当前所有预约记录

5. 输入 `2`
   - 查看统计信息
   - 预期：显示每个自习室的预约数、总可预约槽位数和使用率

6. 输入 `0`
   - 退出管理员端

7. 输入 `0`
   - 退出系统

## 演示时可强调的 C++ 知识点

- `User` 为抽象基类，`Student` 和 `Admin` 体现继承与多态
- `Reservation::operator==` 用于判断预约冲突
- `std::vector` 用于管理学生、管理员、自习室和预约列表
- 文件读写通过 `load*()` / `save*()` 实现持久化
- `SystemManager` 负责统一调度业务逻辑
