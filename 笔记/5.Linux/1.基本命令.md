## 18个基本命令

直接来说记忆思路：


从文件系统的角度出发：
首先我们得有文件对吧！
因此，我们就可以联想到先"创造"文件或者文件夹的操作:
```
mkdir：

mkdir [-p] 文件路径 

[-p]:想要一次性创建2个以上的文件夹时添加


touch:

touch 文件路径 




```


那现在有了文件或者文件夹这些操作对象，我们是不是就可以想着先"查看"文件里的内容：
```
cat:


more:

文件内容太多时，"命令行"可以给我们提供2个操作:
    "空格---翻页
    "Q"----退出

```



接着，又因为，我们的参考对象是文件或者文件夹；
因此，对于文件或文件夹之间就可以进行"复制，移动，删除"操作：
```
cp:

cp [-r] 被复制的文件路径1 复制后的新文件的路径2
注意：
若路径2不存在，操作系统就会在此路径创建一个叫这个名字的"文件或者文件夹"


mv:


rm:



```
