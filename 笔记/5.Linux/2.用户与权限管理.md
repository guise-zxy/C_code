## su用户切换命令
```
su [-] [用户名]
：
切换用户

-:加上则加载环境变量
用户名：不带则默认切换到root
Ctrl+d或者输入exit：回退到上一个用户

注意：
root用户切换到别的用户不需要密码，反之则需要切换


```

## sudo命令
```
sudo 其他命令：赋予这条命令以 root 身份运行

前提：这个普通用户得配置"sudo认证"
sudo认证具体操作：
      a.切换到root用户
      b.输入visudo 或者 vi /etc/sudoers
      c.在文件末尾，添加：
       用户名  ALL=(ALL)   NOPASSWD:ALL      

```
