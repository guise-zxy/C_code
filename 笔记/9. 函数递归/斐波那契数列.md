**如果利用一般迭代方法循环的话，只算大于等于3位置上的斐波那契数（即前提条件为n>=3），那推的时候，循环只用进行n-2次**

```
int main() {

	`int n = 0;`
	
	`int a = 1;`
	`int b = 1;`
	`int c = 2;`
	`scanf("%d", &n);`
	`if (n == 1 || n == 2)`
	`{`
		`printf("1");`
	`}`
	`else`
	`{`

	
		`for (int i = 1; i <= n-2; i++)`
		`{`
			`c = a + b;`
			`a = b;`
			`b = c;`
		
		`}`

		`printf("%d", c);`

    `}
     
```
